## Présentation du projet d'élastoplasticité

On considère une pâle de turbine dans un turboréacteur en régime stationnaire. Cette pâle est modélisée par un système unidimensionnel soumis à un chargement centrifuge modélisé par des forces linéiques variant linéairement avec le rayon courant r.

Cette pâle est supposée constituée de l’alliage de titane TA6V. On suppose que son compor- tement dans la gamme de vitesse considérée est élastoplastique avec un écrouissage isotrope variant en loi de puissance.
